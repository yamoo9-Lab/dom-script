#### DOMScript 디자인 패턴
대상(Target)을 찾아, 이벤트(Event)에 따른 핸들러(Event Handler, function)를 연결(Bind)한다.

#### DOM - Node에 관한 이해
* NodeList (또는 HTML Collection)에 대해서
* 노드리스트는 배열(Array)과 유사하나... 배열은 아니다. (유사배열)

#### 조건문과 반복문 맛보기!
* 조건 확인(조건이 참[true]이면 실행): if()
* 반복적으로 코드 처리(조건이 참[true]일 동안): while()
