# Javascript MVC

애플리케이션 개발 방법론. 데이터와 뷰와 동작을 처리하는 컨트롤러를 각각 섞이지 않게 분리하여 구성하는 방법을 말합니다. Javascript를 사용하여 구조적으로 설계하는 방법을 오늘 최종적으로 공부해봅니다.

- 모델 `Model`
- 뷰 `View`
- 컨트롤러 `Controller`
